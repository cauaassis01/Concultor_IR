# Concultor_IR
Projeto que organiza os dados do usuario de forma que permite a consulta para fazer a declaração do imposto de renda, o projeto possui três planilhas pricipais:
  TITULAR
A planilhas de tirular reune os principais dados relacionado a pessoa civil do usuario, dados como E-mail, telefone, CPF, Nome, etc...
  INFORMES 
A aba de inormes proporciona ao usuario organizar a sua situação financeira através da classificação do banco e declaração de renda.
  NOTAS
Por fim, a aba de notas reune todos os valores de entrada do usuario, sendo classificado por data, categoria podendo ser HOLERITE,CNPJ E FREELANCER. E claro, o valor de entrada.

Essa separação e organização de dados permite que o usuario encontre todos os principais dados e informações necessaria para poder fazer a declaração do imposto de renda, permitindo assim poupar uma grande quantidade de tempo.
